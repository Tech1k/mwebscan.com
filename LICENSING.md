Public License v3.0 (see [LICENSE](LICENSE)).

What that means in practice:

- You may use, study, modify and self-host it freely.
- If you run a modified version as a network service, you must make your
  modified source available to its users (AGPL Section 13). This is the key
  difference from MIT/GPL and is what prevents a competitor from running a
  closed-source SaaS fork.
- A network deployment satisfies Section 13 by offering the Corresponding Source of the
  exact version it runs. The footer link must therefore point to the source of the
  deployed build: this repository if you run it unmodified, or your own published,
  commit-pinned fork if you have changed anything.
